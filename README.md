#API

##Installation
Create the following configuration files (taking example on the samples):

* ```fixtures/config.json```
* ```src/config.json```
* ```doc/src/config.json```

Then launch these commandes:
````npm install
cd doc && npm install && bower install
````

##Test the API
``npm test``

##Launch the API
``npm start``
